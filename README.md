# SUBLIMETAL PRO

Simulador de impresión de alta precisión para sublimación en metal.

## Características

- **35+ plantillas de dispositivos** con medidas reales de fabricantes
- **Vista previa en tiempo real** de tu diseño sobre la placa de metal
- **Control de imagen**: escala, rotación, posición
- **Exportación en alta resolución** (300 DPI)
- **Interfaz moderna** y responsive

## Instalación

```bash
# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev

# Construir para producción
npm run build
```

## Tecnologías

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Lucide React (iconos)

## Estructura del Proyecto

```
src/
├── components/     # Componentes React
├── constants/      # Plantillas de dispositivos
├── hooks/          # Hooks personalizados
├── types/          # Definiciones TypeScript
├── utils/          # Utilidades
├── App.tsx         # Componente principal
├── main.tsx        # Punto de entrada
└── index.css       # Estilos globales
```

## Uso

1. Selecciona un dispositivo de la lista
2. Sube tu diseño (JPG, PNG, WebP, SVG)
3. Ajusta la escala y posición
4. Rota la imagen si es necesario
5. Exporta en alta resolución
